# 个人（家庭）财务报表应用


## 主要内容：

- 实现资产负债表：
    1. 分为资产和负债两个部分
        - 资产部分包括：
            1. 流动资产：现金、活期存款、货币基金等可短期内支取的理财，借给别人已经到期并可随时收回的钱，一句话就是流动性比较好，可以随时拿过来用的钱。 
            2. 投资性资产：包括个人的股票、投资性房产，定期存款，理财型保险，公积金等，一句话就是用来投入并想谋求回报的资产的当前市值。
            3. 自用资产：包括自己的住的房子，汽车，用的贵重家具实物等资产，如果是电子产品汽车类在此录入时要注意折旧，不能按照购买时间的价格录入，一般电子产品4年折旧完成，汽车10年折旧完成。
        - 负债部分包括：
            1. 消费负债：就是借来平时日常消费的钱，这部分钱是肉包子打狗一去不复返的钱，包括消费刷信用卡，消费贷款，借别人的钱用来消费的等。
            2. 投资负债：就是借钱来投资，以谋求增值的钱，包括购买房屋的贷款，或者购买股票时候的融资, 或者借亲戚朋友的钱来创业等。 
            3. 自用资产负债： 用于购买汽车的，用于购买自己住的房子的借来的钱等。
- 实现收入支出表：
    1. 分为收入和支出两个部分
        - 收入部分包括：
            1. 劳动收入（主动收入）：包括各种各样的工作，可以有一个也可以有多个获得的工资收入
            2. 资产收入（被动收入）：比如房屋租金、股票收入、利息等等收入
        - 支出部分包括：
            1. 生活支出：租房租金支出、水电气物业费支出、交通支出、教育支出、基本生活开销支出等等
            2. 资产性支出：固定资产投入、购买汽车支出、房贷还款支出、定期存款支出、活期存款支出、借出金额、金融资产投资（股票购买）
            3. 借款还款： 银行借款本金、银行借款利息、私人借款本金、私人借款利息
- 实现个人（家庭）现金流表：
    主要用来记录每个月的现金流入和流出情况

    **月现金流 = 月总收入 - 月总支出**

    其中：
    月总收入 = 劳动收入 + 资产收入
    月总支出 = 生活支出 + 借款利息 + 其他非本金支出
    🧾 详细说明
    1. 月总收入（现金流入）
    劳动收入（主动收入）：工资、奖金、兼职收入等
    资产收入（被动收入）：房租、股票分红、利息、理财收益等
    例：月工资15,000元 + 房租收入2,000元 = 17,000元
    2. 月总支出（现金流出）
    生活支出：房租/房贷、水电煤、交通、教育、基本生活开销
    借款利息：房贷利息、车贷利息、信用卡利息、私人借款利息
    其他非本金支出：交易手续费、管理费、保险费等
    例：房贷利息2,000元 + 生活支出8,000元 + 交易手续费200元 = 10,200元
    3. 月现金流计算
    月现金流 = 17,000元 - 10,200元 = 6,800元

    **⚠️ 重要说明：避免重复计算**
    资产性支出中的本金部分不计入月支出，因为这属于资产转移而非实际现金流出：
    表格
    项目	|是否计入月现金流	|原因
    ---|---|---
    房贷本金	|❌	|是资产转移，不是现金流出
    股票购买本金	|❌|	是资产转移，不是现金流出
    定期存款本金	|❌|	是资产转移，不是现金流出
    房贷利息	|✅	|是真正的现金流出
    车贷利息	|✅|	是真正的现金流出
    房租支出	|✅|	是真正的现金流出

现金流表部分，需要根据资产负债表以及收入支出表这两个表，自动进行计算和生成。


## 实现功能：

1. 用户登录和管理系统，用户可以进行手机号/邮箱注册、登录，包括个人信息的更改和填写，例如：用户名、性别、年龄、头像等
2. 所有的报表数据都存入数据库，并且根据唯一的用户ID进行管理和存储
3. 用户可以自行个人财务报表、家庭财务报表。（仅能创建一个个人财务报表和一个家庭财务报表）
4. 用户可以在个人或者家庭财务报表下，创建资产负债表和收入支出表（上限各一个）
5. 根据用户的资产负债表和收入支出表，自动计算，并且生成现金流表

## 应用要求：

1. 界面美观，简洁，操作便捷
2. 代码逻辑清晰、注释详细
3. 采用以下技术栈：
    - 前端：使用vue
    - 后端：JAVA语言、springboot框架
    - 数据库：myspl数据库
4. 要求：分别可以在手机安卓使用软件，也可以在windows系统使用
